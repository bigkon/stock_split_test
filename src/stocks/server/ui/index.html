Stock split detection

Usage
Submit csv file with single stock prices history.
File should contain next named columns:
  • date - date, preferably in format of "2020-09-28";
  • open - open price for given date;
  • close - close price for given date;
  • high - highest price for given date;
  • low - lowest price for given date;
  • volume - trade volumes for given date.
After submit chart with price and possible splits will be displayed
Splits will be marked in red circles
